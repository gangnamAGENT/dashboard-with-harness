# 이 폴더에는 CSV 파일을 1개 이상 넣을 수 있습니다.

- 지원 형식: `.csv`
- 권장 인코딩: UTF-8 또는 UTF-8(BOM 포함)

## 입력 규칙

- 모든 CSV에는 헤더 행이 있어야 합니다
- CSV 외 파일이나 폴더는 경고로 남고 분석 대상에서는 빠집니다
- 원본 파일은 수정하지 않습니다

## 여러 CSV를 넣을 때 권장 사항

- 같은 엔터티를 결합하려면 공통 키 컬럼 이름과 의미를 맞춥니다
- 나란히 비교하려면 단위와 기간 축이 같은지 먼저 확인합니다
- 적층하려면 컬럼 의미와 집계 수준이 같은지 맞춥니다
- 서로 독립된 파일이라면 파일명만 봐도 역할이 드러나게 둡니다

그다음 `"파이프라인 실행해줘"` 라고 말하면 됩니다.
